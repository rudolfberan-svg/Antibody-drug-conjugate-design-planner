This html app is an interactive drug discovery tool tailored for planning Antibody–Drug Conjugates (ADCs). It features a guided five-step workflow that allows researchers to select target antigens, configure antibody formats, evaluate cytotoxic payloads, and choose compatible linker chemistries. A dynamic sidebar panel tracks real-time molecular configurations and adjusts a dedicated visual gauge to monitor optimal Drug-to-Antibody Ratios (DAR). Upon completion, the application flags design risks, maps out essential regulatory and preclinical evaluation steps, and allows users to download a comprehensive design report.
